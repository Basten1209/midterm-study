# Week 3 
Topic 1

Should project managers strictly prioritize task specialization for team members over task variety?
Simple & Direct: "Should managers assign people just one type of job to make them experts, or give them many different jobs to keep them interested?"
Professional: "Is it better for project managers to focus on assigning specialized tasks for efficiency, or to ensure team members have a variety of work?"
Casual: "Should bosses keep everyone on the same task forever to get it done fast, or mix it up so they don't get bored?"

YES: Project managers should strictly prioritize task specialization.
NO: Project managers should NOT strictly prioritize specialization (Prioritize Variety).


Topic 2
Is the project manager’s leadership and interpersonal skill set more critical than technical expertise for IS project success?
Simple & Direct: "Is being a good leader and 'people person' more important for a project manager than knowing the technical details?"
Professional: "For an information systems project to succeed, are a manager's leadership and communication skills more valuable than their technical knowledge?"
Casual: "What matters more for a project manager: having great people skills or being a tech wizard?"

YES: Leadership and Interpersonal Skills are More Critical.
NO: Technical Expertise is Equally or More Critical (or the Foundation).
