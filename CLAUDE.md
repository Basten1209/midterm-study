# Midterm Study Repository

4개 대학 과목의 중간고사 준비 자료를 모아둔 저장소.

## 디렉토리 구조

```
├── ergonomics/                      # 인간공학 (Human Factors & Ergonomics)
├── management-infomation-system/    # 경영정보시스템 (MIS, IMEN381)
├── product-operation-management/    # 생산운영관리 (POM)
└── system-analysis-design/          # 시스템분석설계 (SAD)
```

## 자료 유형

| 유형 | 형식 | 설명 |
|------|------|------|
| Lecture | PDF / PPTX | 강의 슬라이드. 주차별로 정리되어 있으며, Part I/II로 나뉘는 경우도 있음 |
| Podcast | m4a | 강의 내용 음성 요약. ENG(영어)와 KOR(한국어) 버전 제공 |
| HW Solution | PDF | 과제 풀이 자료 |
| Showcase | PDF | 특별 주제 발표 자료 |

## 과목별 상세

### 1. ergonomics (인간공학)

- **범위**: 1~6주차
- **자료**: 강의 슬라이드 PDF (각 주차 Part I / Part II), HW 풀이 4건, Showcase 1건
- **주요 토픽**: HFE 소개, 연구방법론, 정보처리, 시각/시각디스플레이 설계, 청각/음성커뮤니케이션

### 2. management-infomation-system (경영정보시스템)

- **범위**: 1~6주차
- **자료**: 강의 슬라이드 PDF
- **주요 토픽**: 오리엔테이션, IS 소개, 글로벌 비즈니스의 IS, 글로벌 e-비즈니스, IS/조직/전략, 운영 우수성 달성, 윤리 및 사회적 이슈

### 3. product-operation-management (생산운영관리)

- **범위**: 1~6주차
- **자료**: 강의 슬라이드 PDF (3~6주차) + 팟캐스트 m4a (1~6주차, ENG/KOR)
- **주요 토픽**: 공급 프로세스 이해, 노동비용 추정/절감, 배칭과 흐름 중단, 운영과 재무의 연결
- **참고**: 1~2주차는 팟캐스트만 존재 (강의 슬라이드 없음)

### 4. system-analysis-design (시스템분석설계)

- **범위**: 1~6주차
- **자료**: 강의 슬라이드 PPTX (3주차) + 팟캐스트 m4a (1~6주차, ENG/KOR)
- **주요 토픽**: IS 프로젝트 관리
- **참고**: 대부분의 주차는 팟캐스트만 존재 (강의 슬라이드는 3주차만 있음)
