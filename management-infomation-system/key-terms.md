# MIS Midterm Key Terms

빈칸채우기 대비용 용어집이다. 약어, 정의, 비교 포인트 위주로 정리했다.

## Week 1: Introduction to MIS

| 용어 | 정의 | 비고 |
|------|------|------|
| **MIS** | 조직의 운영, 관리, 분석, 의사결정을 지원하는 정보시스템 | Management Information Systems |
| **Data** | 가공되지 않은 원시 사실과 숫자 | raw facts |
| **Information** | 의미와 맥락이 부여된 데이터 | decision-oriented |
| **Knowledge** | 정보와 경험이 결합된 이해 및 규칙 | 행동 지침으로 연결 |
| **DSS** | 특정 의사결정을 지원하는 분석 중심 시스템 | Decision Support System |
| **Augmented intelligence** | AI와 인간 판단을 결합하는 의사결정 방식 | Week 1 핵심 결론 |
| **Automation bias** | AI나 자동화 권고를 과신하는 편향 | 인간 판단 약화 |
| **Planning / Organizing / Staffing / Directing / Controlling** | 관리의 다섯 기본 기능 | management functions |

## Week 2: Information Systems in Global Business Today

| 용어 | 정의 | 비고 |
|------|------|------|
| **Digital firm** | 주요 관계, 프로세스, 자산이 디지털 네트워크로 관리되는 기업 | time shifting, space shifting |
| **Operational excellence** | 효율성과 생산성 향상 | 6 strategic objectives 중 하나 |
| **Business model** | 기업이 가치를 만들고 전달하고 판매하는 방식 | wealth creation 방식 |
| **Input / Processing / Output / Feedback** | 정보시스템의 기본 활동 | 정보 생성 흐름 |
| **Information systems literacy** | 기술뿐 아니라 조직과 경영을 포함해 IS를 이해하는 능력 | computer literacy보다 넓음 |
| **Complementary assets** | IT 투자가 실제 가치로 이어지게 하는 보완 자산 | 조직·관리·사회 자산 |
| **Sociotechnical view** | 기술 시스템과 사회 시스템을 함께 최적화하는 관점 | MIS 핵심 관점 |
| **Technical approach** | 수학적 모델과 기술적 설계 중심 접근 | CS, MS, OR |
| **Behavioral approach** | 인간·조직 행동 중심 접근 | sociology, psychology, economics |

| 용어 | 정의 | 비고 |
|------|------|------|

| **ESS** | 고위 경영진용 전략 지원 시스템 | Executive Support System |

| **ERP** | 내부 기능과 프로세스를 통합하는 전사 시스템 | Enterprise Resource Planning |
| **SCM** | 공급망 전체를 조정하는 시스템 | Supply Chain Management |
| **CRM** | 고객 관계를 통합 관리하는 시스템 | Customer Relationship Management |
| **KMS** | 조직 지식의 생성·공유·활용을 돕는 시스템 | Knowledge Management System |

| **Extranet** | 외부 파트너에게 일부 개방된 네트워크 | 공급자/고객 연결 |
| **CIO** | 정보시스템 기능을 총괄하는 최고책임자 | Chief Information Officer |
| **IT governance** | IT 의사결정 권한과 책임의 구조 | alignment와 연결 |

## Week 4: Information Systems, Organizations, and Strategy

| 용어 | 정의 | 비고 |
|------|------|------|
| **Routines** | 반복적으로 수행되는 규칙과 절차 | 프로세스의 구성 단위 |
| **Organizational culture** | 조직의 공유 가치와 암묵적 규범 | 시스템 수용성에 영향 |
| **Organizational politics** | 이해관계 충돌과 권력 작용 | IS 도입 저항 원인 |
| **Disruptive technology** | 기존 시장과 경쟁 구조를 바꾸는 기술 | fast follower도 중요 |
| **Porter’s competitive forces model** | 산업 경쟁 압력을 설명하는 틀 | 전략 수립용 |
| **Value chain** | 기업의 가치창출 활동 사슬 | primary / support activities |
| **Benchmarking** | 타 기업 또는 기준 대비 성과 비교 | best practice 식별 |
| **Value web** | IT로 연결된 기업 네트워크 | 빠른 조정 가능 |
| **Synergy** | 사업부 간 결합으로 추가 성과 창출 | shared inputs/outputs |
| **Core competency** | 경쟁사가 쉽게 모방하기 어려운 핵심 역량 | 지식 축적 기반 |
| **Network economics** | 사용자 증가에 따라 가치가 증가하는 경제성 | positive network effects |
| **Virtual company** | 디지털 네트워크 기반 경계 없는 기업 모델 | outsourcing과 연결 |
| **Business ecosystem** | 상호의존적 기업 네트워크 | keystone firm 존재 |
| **Transaction cost theory** | 거래비용 최소화 관점의 조직 설명 | IT가 거래비용 감소 |
| **Agency theory** | 감독과 통제 비용 관점의 조직 설명 | middle management 감소 |
| **Alignment** | IT와 비즈니스 목표의 정렬 | 수익성과 직결 |

| 용어 | 정의 | 비고 |
|------|------|------|
| **Enterprise system** | 전사 정보를 통합하는 시스템 | ERP와 연결 |

| **Bullwhip effect** | 작은 수요 변화가 공급망 상류에서 증폭되는 현상 | SCM 핵심 개념 |
| **Supply chain planning** | 수요예측과 생산·재고 계획 기능 | planning system |
| **Supply chain execution** | 실제 물류 흐름과 배송 관리 기능 | execution system |
| **Push-based model** | 예측 기반 공급 모델 | forecast-driven |
| **Pull-based model** | 실제 주문 기반 공급 모델 | demand-driven |
| **Touch point** | 고객과 기업이 접촉하는 채널 | 전화, 메일, SNS 등 |
| **SFA** | 영업활동 자동화 기능 | Sales Force Automation |
| **Operational CRM** | 고객 접점 중심 CRM 애플리케이션 | sales/service/marketing |
| **Analytical CRM** | 고객 데이터 분석 중심 CRM | data warehouse 기반 |
| **Churn rate** | 고객 이탈률 | CRM 성과 지표 |
| **PRM** | 파트너 관계 관리 모듈 | Partner Relationship Management |
| **ERM** | 직원 관계 관리 모듈 | Employee Relationship Management |

## Week 6: Ethical and Social Issues in Information Systems

| 용어 | 정의 | 비고 |
|------|------|------|
| **Ethics** | 옳고 그름을 판단하는 원칙 | 행동 선택 기준 |
| **Information rights and obligations** | 정보에 대한 권리와 의무 | moral dimension |
| **Property rights and obligations** | 지식재산 관련 권리와 의무 | moral dimension |
| **Accountability and control** | 책임 추적과 통제 구조 | moral dimension |
| **System quality** | 데이터와 시스템 품질 기준 | moral dimension |
| **Quality of life** | 기술이 삶에 미치는 영향 | moral dimension |
| **Responsibility** | 자신의 선택 결과를 감수하는 것 | 개인 차원 |
| **Accountability** | 누가 행동했는지 추적 가능한 상태 | 제도 차원 |
| **Liability** | 법적 책임과 손해배상 가능성 | legal domain |
| **Due process** | 규칙의 공정한 적용 절차 | appeal 가능성 |
| **GDPR** | EU의 강력한 개인정보보호 규정 | right to be forgotten 포함 |
| **Cookie** | 사용자 브라우징 기록용 작은 파일 | personalization과 tracking |
| **Web beacon** | 웹/이메일 내 추적용 프로그램 | web bug |

| **Patent** | 발명에 대한 독점 권리 | 보통 20년 |
| **Trademark** | 상품·서비스 식별 표지 | 브랜드 식별 |
| **Trade secret** | 비공개 사업 지식 보호 | NDA 필요 |
| **DMCA** | 디지털 저작권 보호를 위한 미국 법 | piracy 대응 |
| **Digital divide** | 기술 접근 격차 | 사회적 불평등 연결 |
| **RSI / CTS / CVS** | IT 사용 관련 건강 문제 | 반복손상 / 수근관 / 시각증후군 |

## Week 7: Telecommunications, the Internet, and Wireless Technology

| 용어 | 정의 | 비고 |
|------|------|------|
| **NID** | 컴퓨터를 네트워크에 연결하는 장치 | Network Interface Device |
| **NOS** | 네트워크 통신과 자원을 관리하는 운영체제 | Network Operating System |
| **Hub** | 수신 데이터를 모든 장치에 보내는 장비 | 단순 연결 |
| **Switch** | 데이터를 특정 목적지로 전달하는 장비 | hub보다 효율적 |
| **Router** | 다른 네트워크로 패킷을 전달하는 장비 | Internet 연결 핵심 |
| **SDN** | 소프트웨어 기반 중앙 제어 네트워크 | Software-Defined Networking |
| **Client/server computing** | 서버-클라이언트 분산 컴퓨팅 모델 | modern network foundation |
| **Packet switching** | 메시지를 패킷으로 분할해 전송하는 방식 | circuit switching 대비 효율적 |
| **TCP/IP** | 인터넷 통신의 표준 프로토콜 집합 | reliability + routing |
| **LAN / MAN / WAN** | 근거리 / 도시권 / 광역 네트워크 | 범위별 구분 |
| **Modem** | 디지털·아날로그 신호 변환 장치 | modulator-demodulator |
| **Bandwidth** | 통신 채널의 총 전송 용량 | 넓을수록 빠름 |
| **ISP** | 인터넷 접속을 제공하는 사업자 | Internet Service Provider |
| **IP address** | 네트워크 상 장치의 주소 | Internet addressing |
| **DNS** | 도메인 이름을 IP 주소로 변환하는 시스템 | Domain Name System |
| **IAB / ICANN / W3C** | 인터넷 거버넌스 관련 기관 | 표준·주소·웹 규격 |
| **IPv6** | 128-bit 기반 차세대 IP 주소 체계 | IPv4 주소 부족 해결 |
| **FTP** | 파일 전송용 인터넷 서비스 | File Transfer Protocol |
| **VoIP** | IP 기반 음성 통신 | Voice over IP |
| **VPN** | 암호화된 사설 네트워크 | Virtual Private Network |
| **HTML / HTTP / URL** | 웹 문서 형식 / 전송 규약 / 자원 위치 | Web 핵심 3요소 |
| **Deep web / Dark web** | 일반 검색엔진 미색인 웹 / 숨겨진 일부 영역 | 검색 파트 핵심 |
| **SEO / SEM** | 자연검색 최적화 / 검색엔진 마케팅 | search marketing |
| **Bluetooth** | 근거리 저전력 무선 연결 표준 | 802.15 |
| **Wi-Fi** | 무선 LAN 표준 | 802.11 |
| **WiMax** | 광역 무선 브로드밴드 표준 | 802.16 |
| **RFID** | 무선 주파수 식별 기술 | 물류 추적 |
| **NFC** | 초근거리 무선 통신 기술 | 모바일 결제 |
| **WSN** | 센서 노드 기반 무선 네트워크 | IoT와 연결 |
